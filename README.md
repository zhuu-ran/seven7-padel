# Seven7 Padel

Website booking lapangan padel berbasis PHP dan MySQL.

## Fitur
- Login & Register
- Venue Booking
- Booking Lapangan dengan slot waktu
- Dashboard Admin (CRUD Lapangan, Slot, Booking, User)
- Open Play / Event

## Cara Install
1. Clone repository ini
2. Import `padel_booking.sql` ke phpMyAdmin
3. Sesuaikan `config/db.php` dengan kredensial database kamu
4. Jalankan di localhost dengan XAMPP

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
